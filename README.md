# Análise de Viagens Corporativas

Projeto inspirado em experiência real de otimização e renegociação de viagens corporativas.

## Problema
Empresas frequentemente possuem custos elevados com passagens e hospedagens por falta de análise estruturada dos dados de viagem.

## Objetivo
Analisar dados de viagens corporativas para identificar padrões de gasto, oportunidades de economia e melhorias de processo.

## Possíveis análises
- Custo médio por viagem
- Companhias aéreas mais utilizadas
- Rotas mais frequentes
- Comparação de custos por fornecedor

## Ferramentas utilizadas
Excel  
SQL (em desenvolvimento)

## Contexto
Projeto inspirado na iniciativa **Soma Viagens**, voltada à centralização e renegociação de viagens corporativas com apoio de dados.
